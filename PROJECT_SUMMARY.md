```bash
git clone https://github.com/YOUR_USERNAME/apollo-agriguard.git
cd apollo-agriguard
git add .
git commit -m "Apollo AgriGuard premium website"
git push origin main

```

### Option 2: Vercel (One-Click)
```bash

# Site live in < 1 minute
# Auto-deploys on git push
```

### Option 3: Traditional Hosting
```bash
scp -r apollo-website/* user@server:/var/www/html/
# Add .htaccess for security & performance
# Enable HTTPS/SSL
```

**Don't know which one?** Start with GitHub Pages - it's free and fast!

## 🎨 Customization Quick Start

### Change Company Name (5 seconds)
Find & replace: "Chatake Innoworks" → "Your Company"
Find & replace: "Apollo AgriGuard" → "Your Product"

### Change Colors (2 minutes)
Edit `css/premium-style.css` `:root` section:
```css
--primary-dark-green: #0D5F3F;  → Your primary color
--accent-copper: #D4A574;        → Your accent color
```

### Add Your Logo (1 minute)
- Save logo as `assets/logos/logo.svg`
- Update HTML to reference it
- That's it!

### Add Images (5 minutes)
- Drop images in `assets/images/`
- Update HTML `src` attributes
- Responsive design handles sizing automatically

### Update Contact Info (2 minutes)
- Find phone number: "+91 9876 543 210"
- Find email: "apollo@chatakeinnoworks.com"
- Replace with your info
