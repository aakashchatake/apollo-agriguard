## 🚀 Quick Start

### Development
1. Clone the repository
2. Open `index.html` in a modern browser
3. Use live server for development:
   ```bash
   python -m http.server 8000
   # or
   npx http-server
   ```
4. Visit `http://localhost:8000`

### Customization
1. **Colors**: Edit CSS variables in `premium-style.css` `:root`
2. **Content**: Edit HTML files directly
3. **Animations**: Modify keyframes in `animations.css`
4. **Scripts**: Add custom logic to `main.js`
